Test data are not available in database.

We need:

  1. Test with name background
  2. Goal with name buttonPressed
  3. Three experiments red, blue and green
Goal and experiments needs to be added to Test; test has to be active.

You can add these objects, using the admin (recommended to learn things), or press the button to generate test data.

This is, what will happen behind the scenes (in the view): 

test = Test.objects.create(name = "background", active = True)
test.goals.add(Goal.objects.create(name = "buttonPressed"))
test.experiments.add(Experiment.objects.create(name="red"))
test.experiments.add(Experiment.objects.create(name="blue"))
test.experiments.add(Experiment.objects.create(name="green"))
test.save()
